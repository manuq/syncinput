Module: Input

Gamepad provides basic support for gamepads. Some gamepads require a button press to being detected. Gamepad implementation across browsers is still fragmented, every browser implements it a bit differently, so test it on every target before deploying an application using it. For more information about the Gamepad API state take look at the W3C Gamepad API page https:// www.w3.org/TR/gamepad/.
